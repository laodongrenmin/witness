### 环境
#### 1 python版本
- python 3.7.6
```
 安装命令：conda create -n python37 python=3.7
```
### 2 依赖包
- watchdog 
```
 安装命令 pip install watchdog
```

- libvips
```
1. 下载 vips解压就可以用了，官网特别慢，csdn上下载的  https://github.com/libvips/libvips/releases  vips-dev-w64-all-8.9.1.zip
2. 安装pyvips  pip install pyvips
```
